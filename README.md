# Online-Food-Ordering-System-Python-Flask-
Online Food Ordering System is a basic web app built using Python Flask and SQLite. Users can browse food items, place orders, and download PDF receipts. An admin panel manages menu items and orders. This project helped for learn Flask basics and database operations.
